# ✋ AI Sign Language Recognition System

---

# 📌 Overview

The AI Sign Language Recognition System is a Deep Learning and Computer Vision project that recognizes sign language hand gestures and predicts the corresponding alphabet.

This system demonstrates how CNNs (Convolutional Neural Networks) can be used for image classification and accessibility-focused AI applications.

---

# 🎯 Problem Statement

People with hearing or speech disabilities use sign language for communication, but many people do not understand sign language.

This project helps bridge communication gaps using AI-powered gesture recognition.

---

# ✨ Features

- Upload hand sign image
